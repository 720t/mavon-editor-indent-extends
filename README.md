# mavon-editor-indent-extends

mavon-editor 编辑器 多行缩进组件

# 克隆项目

```bash
git clone https://github.com/720t/mavon-editor-indent-extends.git
```

## 启动服务

```
npm run dev
yarn dev
```

浏览器访问 [http://127.0.0.1:9528](http://127.0.0.1:9528)

## 发布

```bash
# 构建测试环境
npm run build:stage

# 构建生产环境
npm run build:prod
```

## 其它

```bash
# 预览发布环境效果
npm run preview
yarn preview

# 预览发布环境效果 + 静态资源分析
npm run preview:report
yarn preview:report

# 代码格式检查
npm run lint
yarn lint

# 代码格式检查并自动修复
npm run lint:fix
yarn lint:fix
```
